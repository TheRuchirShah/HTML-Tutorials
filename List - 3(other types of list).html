Other Types of List

The other types of list apart from Order-List & Unorder List are

  • Description List
  • List Style Position - (list-style-position)
  • Image List - (list-style-image)

Example - Description List

Used to create a list of terms and their descriptions.

An unordered list starts with the <dl> tag and end with </dl>. Each list item starts with the <li> tag. and end with </li>. The <dt> specifies term end with </dt>. And <dd> specifies definition end with </dd>
